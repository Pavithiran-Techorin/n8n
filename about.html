About This Platform

This is a private, self-hosted instance of n8n - a powerful workflow automation platform that helps connect different services and automate repetitive tasks.

What is n8n?

n8n is an open-source workflow automation tool that allows you to connect various services and create automated workflows. It provides:

  • 400+ integrations with popular services
  • Visual workflow designer
  • Advanced AI capabilities with LangChain integration
  • Secure, self-hosted deployment

Security & Privacy

This platform is designed with security and privacy in mind:

  • All data is processed locally on our servers
  • No third-party data sharing
  • Encrypted data storage
  • Regular security updates

AI Integration

This instance includes local AI capabilities through:

  • Ollama - Local LLM processing
  • Qdrant - Vector database for semantic search
  • LangChain integration for advanced AI workflows
